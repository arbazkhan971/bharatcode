# Installing BharatCode

## Homebrew (macOS / Linux)

BharatCode is distributed through a [Homebrew tap](https://github.com/arbazkhan971/homebrew-tap).

```sh
brew install arbazkhan971/tap/bharatcode
```

This pulls the formula from the `arbazkhan971/homebrew-tap` repository (Homebrew
expands `arbazkhan971/tap` to `arbazkhan971/homebrew-tap`) and installs the
`bharatcode` binary onto your `PATH`.

To upgrade later:

```sh
brew upgrade bharatcode
```

Verify the install:

```sh
bharatcode version
```
